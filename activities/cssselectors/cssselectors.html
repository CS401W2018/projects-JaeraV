List out the different ways to select an item.

  1. Tag name
  2. Descendant
  3. ID
  4. Class

What is the difference between an ID and a class?

When using an ID, you can select any tag but it can only be used once. Every tag can only have one and they usually use nouns. Also, you need to use "#". On the other hand, when using a class, it is almost the same with ID but you can use class names as much as you want. A tag can have more than one class and they usually use adjective. Also, you need to use a "."

What is the difference between a SPAN tag and a DIV tag?

A SPAN tag is for inline styling and it doesn't create line breaks. While DIV tag is used for grouping elements and styling the whole block.

What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

A DIF tag is used for styling a block while the HEADER, MAIN, FOOTER tags are used to provide better structure for your HTML document because it makes the different parts in your document clear and understandable.
