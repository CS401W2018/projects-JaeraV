Engagement Questions:

Why is the viewport meta tag important?

It is important because the viewport meta tag ensures that the web pages you make is displayed correctly on any size of screen like on mobile phones or a bigger screen.

How can you adjust the CSS for printing?

You can adjust the CSS for printing by making another CSS file and then linking the CSS file for printing to your HTML document with media="print" inside the head tag.

When would you want to adjust the CSS for printing?

You would want to adjust the CSS for printing when you want your webpage to look professional when users want to print it. You will also want to make it if you want to make adjustments to hide some unnecessary elements like the nav links so that it won't appear when printing.
