Engagement Questions:

The form tag has an "action" attribute. What does it do?

It specifies the link or URL where the data will go once the user click submit.

The form tag has a "method" attribute. What does it do?

It defines how the data will be sent to the server. It is how the web server and the webpage interact when the form is submitted.

What is the difference between a "POST" and a "GET"?

One difference is that using "GET" will make your data visible in the URL while "POST" will not show the data in the URL. "GET" is usually used when you want to retrieve or get something, for example, clicking on a link or when you are doing a search. On the other hand, "POST" is when you want to change stuffs in the server, for example, creating an account, submitting or adding something.
