Requirement 2

Explain the difference between an absolute link and a relative link. Why would you use relative links?

When using absolute link, you can link to other websites or your own. On the other hand, when using relative link, you can link stuffs from your own website only. We use relative links on our website so that the links would still work no matter where the website is hosted. If your website moves to a different domain name, the relative links will still work.
