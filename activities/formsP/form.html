What is JavaScript? How did it get its name?

JavaScript runs inside the web browser. It is a programming language which is used to make web pages interactive. It was created in 1995 by Brendan Eich and was called "Live Script" then it became "JavaScript". The name is similar to the programming language, Java, but they are very different.

What are two ways you can run JavaScript?

In the Browser (DOM) and Node.js

What was the most popular front-end js library?

jQuery.js

What is the most popular front-end js library today?

React.js
