Engagement Questions:

Why do we need a viewport meta tag?

We need the viewport meta tag because it makes the media queries to work. It is important because it makes our webpage look good and work on any screen size.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The last thing you say is what takes priority. If "flex-direction: column;" is followed by "flex-direction: row;", then "flex-direction: row;" will be prioritized.

How do you change the CSS for a web page displayed on a phone?

You can change it my inserting a media query on your CSS file, "@media only screen and (max-width: 800px){ insert changes here}"
