Engagement Questions:

Why don't we use the border property?

We don't use the border property because CSS can't overwrite or change it. Also, there will be a validation error with your HTML.

What is the easiest way to color every other row a different color?

By using CSS selectors and using odd or even for the alternation effect. For example, ".schedule tbody tr:nth-child(even)"

When should we use a table?

We should use a table when we want to present a data in our webpage. We do this so we can present our data in a more organized way for the users.
