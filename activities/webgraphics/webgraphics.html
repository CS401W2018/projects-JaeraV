Engagement Questions

What kind of photos are good to encode using JPEG?

Photos that have a lot of colors and details are good to encode using JPEG because it uses “lossy compression,” meaning it reduces the file size, which slightly lowers the quality of the photos.

What kind of photos are good to encode using GIF?

Photos with limited colors or have simple graphics or animations are good to encode using GIF because it supports animation and use “lossless compression.”

What is unique about the PNG format?

PNG format is unique because it supports transparency, meaning it has the ability to support transparent backgrounds.

How do you resize your photo to the size it should be displayed?

We can resize our photo by using different image resizing tools like photoshop, Microsoft paint for windows, preview for Mac, using CSS or even some tools online.

Why should you resize photos using photo editing software instead of resizing it using CSS?

It is better to resize photos using photo editing software instead of using CSS because it is for better performance of your website. When you use a large picture for your website, it might take a lot of megabytes to transfer over the network which might slow down your website. It is okay to resize using CSS but it should only be for minor adjustments.
