Questions

What is the difference between block tags and inline tags.

Block tags start a new line and it has automatic margin before and after. Some examples are the paragraph tag, header, footer, main and many more. On the other hand, inline tags does not start a new line. They are inline or go with the text. Some examples are the anchor link tag, image tag, phrase tags like emphasis, strong, mark, and many more.

What is the margin, border, and padding?

Margin pushes away the box from other boxes nearby. Border is the box itself, you can style it with a thin or heavy border. Lastly, padding pushes the content inside of the box away from the edge.

What are units of measurement you can use in CSS?

We can use pixels, fixed measurements like mm, cm, and relative measurements like percentages.

When would you use a figure tag?

When we have images, figures, diagrams or illustrations and we want to put it in a block tag or we want to put a caption using figcaption tag. What this figure tag does is it surrounds our image or figure and it becomes a block tag.
