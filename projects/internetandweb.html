Internet

A global network where computers, devices or electronic devices are interconnected. It allows the exchange of information between computers and other devices which makes online communication possible.

Web

This is a system of interconnected documents or media that can be accessed through the internet. Usually these documents can be accessed using web browsers.

How are the Internet and Web different?

The internet is a network that connects computers and other electronic devices, while the web allows people to access information or exchange information with the use of the internet.

ASCII and Unicode

ASCII is a system that helps computers to understand and display the text, letters, numbers and symbols by using specific numbers assigned to it by ASCII. Every character in the keyboard like “A”, “0”, or “!” are represented by a specific number. It can support 128 characters or 256 in the extended version. On the other hand, Unicode is a system that assigns a unique number for every character from any language or emojis. Unlike ASCII, unicode can handle thousands of different characters from different languages.

What is UTF-8

UTF-8(Unicode Transformation Format - 8 bit) is a way to encode Unicode characters in a way that it can be stored and transferred or transmitted by computers. It is efficient to use because it saves space while supporting a wide range of characters.

URL

URL (Uniform Resource Locator) is the address that is used to access resources available in the internet. It tells the web browser to find and retrieve the content you are searching for like web pages, videos or images.

Domain Name

It is the address used to locate or visit a website in the internet. For example, “google.com”

HTTP

HTTP (Hypertext Transfer Protocol) is a protocol for the transmission of information in the internet. It defines how messages or information are formatted and transmitted between a web server and a client.

HTTPS

HTTPS (Hypertext Transfer Protocol Secure) is an extension of the HTTP where in the data are transferred over the web in an encrypted form. It ensures that the communication between the web server and the client is secure.

IP Address

IP Address (Internet Protocol Address) is a unique numerical label assigned to devices connected to a network. It enables devices to locate each other within a local network or over the internet.

DNS

DNS (Domain Name System) is a system that translates domain names into IP addresses so that the web browser can locate and connect to the website’s server.

W3C Recommendations

W3C (World Wide Web Consortium) Recommendations are standards and guidelines that ensures the development of web technologies to be accessible, cohesive and functional platform for everyone.

XML, XHTML, HTML, HTML5

XML (eXtensible Markup Language) is designed to store and transport data and is not intended for presentation.
XHTML (eXtensible Hypertext Markup Language) is a more defined version of HTML. It follows a more standardized and stricter rules than the normal HTML.
HTML (Hypertext Markup Language) is a standard markup language that is used in creating and designing web pages and applications.
HTML5 is the latest version of HTML. It has new features and improvements for creating better web development or web applications.
